Compétence 5

Identifier les besoins métiers des clients et des utilisateurs

Déroulé & explication du projet

Étapes du projet :

  1. Organisation d'équipe : répartition du travail, rôles....
  2. Compréhension et rédaction des besoins
  3. Réalisation de l'arborescence
  4. Création de la charte graphique
  5. Maquettage
  6. Programmation avec outil de versionning
  7. Présentation orale en équipe

Contexte

Projet SAÉ S1.05 - Recueil des besoins pour un site web: Conception et réalisation en équipe d'un site web dédié aux JO Paris 2024 suite à une interview avec Tony Estanguet. Le projet débute par l'expression des besoins client, puis la personnalisation avec le choix d'une catégorie sportive (natation vitesse dans votre cas). L'équipe doit créer l'arborescence du site, définir le contenu des pages (Accueil, Discipline, Épreuve, Portrait d'un sportif), établir une charte graphique harmonisée avec l'univers olympique, réaliser les maquettes responsive (smartphone/desktop), puis développer le site en HTML/CSS/JavaScript avec une approche responsive native sans framework.

Technologies utilisés

  • Visual Studio Code
  • Google DRIVE
  • Figma
  • Langage HTML
  • Langage CSS
  • Langage Java Script
