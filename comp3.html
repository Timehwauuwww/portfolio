Compétence 3

Installer et configurer un poste de travail

Déroulé & explication du projet

Étapes du projet :

  1. Installation d’un poste pour le développement : mise en œuvre d'une chaîne de processus (workflow) s'intégrant dans un environnement de développement.
  2. Identifier les outils nécessaires
  3. Préparation des fichiers de test (donnés et configs)
  4. Proposition d'un workflow enchaînant les outils
  5. Production finale et "scriptage" du workflow

Contexte

Projet SAÉ S1.03 - Installation d'un poste de développement: Développement en équipe d'une chaîne de traitement automatisée pour les fichiers du site Web des JO Paris 2024. L'équipe doit analyser différents types de fichiers (images, textes, tableaux Excel), les convertir aux formats requis (WebP, CSV, texte structuré) en utilisant Docker et des outils comme ImageMagick. Le projet inclut le traitement automatisé des médailles olympiques avec calcul de totaux, tri par classement et téléchargement dynamique de drapeaux, ainsi que la génération d'un PDF final du tableau des médailles.

Technologies utilisés

  • Google DRIVE
  • Langage Bash
  • Langage PHP
  • Docker
  • Terminal
  • Langage HTML
  • Langage CSS
