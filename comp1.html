Principe du 2048

Explications du jeu :

  • Le jeu se joue sur une grille 4×4
  • Il commence avec deux tuiles (généralement de valeur 2)
  • Le joueur déplace toutes les tuiles simultanément vers l'une des quatre directions (haut, bas, gauche, droite)
  • Après chaque mouvement, une nouvelle tuile (de valeur 2 ou 4) apparaît aléatoirement sur la grille
  • Quand deux tuiles de même valeur se touchent lors d'un déplacement, elles fusionnent en une seule tuile dont la valeur est la somme des deux
  • L'objectif est d'obtenir une tuile de valeur 2048 (d'où le nom du jeu)
  • Le jeu se termine quand il n'y a plus de mouvements possibles (grille pleine sans fusions possibles)

Contexte

Projet scolaire - 2048 avec interface graphique : Développement en binôme d’un jeu 2048 en Python durant l’année de Terminale. Le joueur fusionne des tuiles contenant des puissances de 2 pour atteindre le score maximal. Le projet intègre une interface graphique réalisée avec la bibliothèque Kivy pour une meilleure expérience utilisateur. L’objectif était de reproduire fidèlement les règles du jeu original tout en explorant la gestion d’événements, l’affichage dynamique et la logique de fusion. Le développement s’est fait en plusieurs étapes pour tester et améliorer progressivement les fonctionnalités du jeu.

Technologies utilisés

  • Visual Studio Code
  • Google DRIVE
  • Langage python
  • Trello
